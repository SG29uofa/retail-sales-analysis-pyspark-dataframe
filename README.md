# Retail Sales Analysis using PySpark DataFrames
A PySpark DataFrame project analyzing retail sales data using filtering, aggregation, joins, and Spark SQL to generate business insights.

## Objective
This project analyzes a retail sales dataset using PySpark DataFrames to extract business insights through filtering, aggregation, joins, and SQL queries.

---

## Dataset
The dataset contains retail transaction data with the following columns:

- Order ID
- Order Date
- Customer ID
- Customer Name
- Segment
- City
- State
- Region
- Category
- Sub-Category
- Product Name
- Sales

---

## Technologies Used
- Python
- PySpark
- Apache Spark (DataFrame API)

---

## Key Operations
- Loaded CSV data into Spark DataFrame
- Performed data exploration and schema validation
- Converted Sales column to numeric type
- Applied filtering on high-value transactions
- Created new derived columns
- Used groupBy and aggregation functions
- Performed sorting to identify top categories and regions
- Implemented joins between DataFrames
- Executed SQL queries using Spark SQL

---

## Key Insights
- Sales distribution varies significantly across categories
- Certain regions contribute more to total revenue
- High-value transactions highlight premium sales patterns
- Customer distribution differs by region and segment

---

## Learning Outcomes
- DataFrame transformations and actions
- Aggregation using groupBy
- Filtering and column manipulation
- Joining DataFrames
- Using Spark SQL for analysis
- Handling data types and schema issues
